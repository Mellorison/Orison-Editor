Changes

2.1.0.4

August 18, 2019

Fixes

  • Fixed Shift Renamer crash
  • Fixed tilemaps with XML export modes not being salvageable if the level size is wrong
  • Fixed tilemaps not drawing at half alpha above the current layer
  • Fixed crash on level resize in some cases involved tile layers
  • Reversed the order of layers in the layer window so it makes sense now

Fixes

2.1.0.0

August 10, 2019

Changes

  • Rewrote rendering code to remove XNA from Orison Editor!
  • Various visual improvements
  • Entities can now be both tiled and rotatable
  • Added grid layer export option: Grid Rectangles
  • Better and More descriptive salvaged level options
  • Lots of optimizations, particularly:
    • All tile layer actions
    • Entity move and scale actions
  • View changes:
    • Zooming now scales at the mouse position
    • Can now press the HOME key to center the level view
    • Can now press +/- to zoom in/out
  • Entity selection tool:
    • Right-click now deselects
    • CTRL+select now toggles selection on entity
    • Copy/paste now ends with all pasted entities selected
  • Added level batch utilities:
    • Batch Resaver - resave levels to quickly apply project changes
    • Shift Renamer - rename levels to help rearrange their numerical order
    • Swap Renamer - quickly swap the filenames of two level files
  • Added this changelog to the start page
  • Added website/donate buttons to the about window and start page

Fixes

  • Fixed tile palette sometimes being blank on startup
  • Fixed tile layers not drawing if the graphics card resets
  • Fixed tileset dropdown sometimes capturing WASD input
  • Tile selector now returns focus to the level editor on mouse selection
  • Fixed tile selector not applying tile separation correctly
  • Tile tools now correctly apply multi-tile selections
  • Entity previews in the entity selector now crop correctly
  • Fixed editor sometimes crashing on entity paste
